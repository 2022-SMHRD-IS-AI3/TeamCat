꺼냉 사용법

"꺼냉"은 이웃 간 물건 대여 서비스 플랫폼 입니다.

내 주변에서 내 이웃에게 편하게 물건을 빌려보세요!

STEP1. 대여 물품 선택
내가 필요한 물건을 선택하세요.

tip! 원하는 물건을 찾기 위해 검색 기능을 사용해 보세요!

STEP2. 메시지 보내기
메시지를 보내 대여자와 대여 날짜와 시간을 조율해 보세요.
대화를 나눌 때 서로 예의를 지켜주세요!

tip! 반납 시간을 미리 정하면 반납이 더욱 쉬워요.

STEP3. 예약하기
조율이 끝났다면 예약하기 버튼을 사용해 예약을 진행하세요.
대여자가 예약 확인 버튼을 누르면 예약이 가능해져요.
STEP4. 반납하기
대여일이 끝나면 반납을 진행하세요.
반납이 끝나면 꼭 반납완료 버튼을 클릭하세요!
